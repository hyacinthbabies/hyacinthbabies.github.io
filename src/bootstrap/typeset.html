加mark标记：You can use the mark tag to highlight text.

加del删除：This line of text is meant to be treated as deleted text.

加s标记不用的：This line of text is meant to be treated as no longer accurate.

加ins标记引入：This line of text is meant to be treated as an addition to the document.

加u表示下划线：This line of text will render as underlined

加small变小：This line of text is meant to be treated as fine print.

加strong加粗：rendered as bold text

加em斜体：rendered as italicized text

加code标记代码(注意转义)：For example, <section> should be wrapped as inline.

加kbd表示键盘输入：To switch directories, type cd followed by the name of the directory.

加pre标记一段代码，加.pre-scrollable类垂直滚动：

加var表示变量：y = mx + b

加samp表示程序输出：This text is meant to be treated as sample output from a computer program.
